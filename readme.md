## 📌 Sobre o Projeto
Este laboratório tem como objetivo aplicar conceitos de **Infraestrutura como Código (IaC)** na prática, utilizando o Terraform para criar e gerenciar recursos no Azure de forma automatizada e versionada.

## 🧰 Tecnologias Utilizadas
- Terraform
- Microsoft Azure
- Git & GitHub

## 🎯 Objetivo
- Automatizar provisionamento de infraestrutura
- Aplicar boas práticas de DevOps
- Criar ambientes reprodutíveis na nuvem
